# 🦀 Rust Survivor

Динамичный 2D-экшен в жанре «survival roguelike», разработанный на языке **Rust** с использованием игрового движка **Bevy**. Проект демонстрирует мощь ECS-архитектуры (Entity Component System), работу с процедурной генерацией врагов и бесконечным игровым миром.


## 🛠 Технический стек

* **Движок:** [Bevy 0.18.1](https://bevyengine.org/) (последняя стабильная версия).
* **Язык:** Rust (Edition 2024).
* **Платформы:** Windows, Linux, macOS, а также поддержка **WebAssembly (WASM)**.
* **Математика и Рандом:** `rand` с поддержкой `wasm_js`.

## ✨ Ключевые особенности

### 1. Модульная ECS Архитектура
Код разделен на независимые плагины, что обеспечивает высокую читаемость и масштабируемость:
* **CorePlugin:** Управление камерой, мировыми координатами и бесконечной сеткой фона.
* **PlayerPlugin:** Обработка ввода пользователя, физика движения и механика прицеливания.
* **EnemyPlugin:** ИИ преследования, система спавна и алгоритм разделения (предотвращение накладывания врагов друг на друга).
* **CombatPlugin:** Система стрельбы, расчет траекторий снарядов и обработка коллизий.

### 2. Бесконечный игровой мир
Реализована система **World Wrap**: когда игрок или объекты выходят за пределы `AREA_SIZE`, они мгновенно переносятся на противоположную сторону, создавая иллюзию бесконечного пространства.

### 3. Механики геймплея
* **Динамический спавн:** Враги появляются группами по радиусу вокруг игрока.
* **Soft Collisions:** Враги «расталкивают» друг друга, обеспечивая органичное поведение толпы.
* **Система здоровья:** Реализован базовый расчет урона как для игрока, так и для противников.

## 🚀 Быстрый запуск

### Предварительные требования
Убедитесь, что у вас установлен Rust (рекомендуется версия 1.80+).

### Запуск Desktop версии
```bash
cargo run --release
```

### Сборка под Web (WASM)
Проект настроен для работы в браузере с использованием `wasm-bindgen` или `trunk`:
```bash
trunk serve
```


## 📂 Структура проекта

* `Core`: Инициализация рендеринга и системных параметров.
* `Player`: Логика сущности игрока.
* `Enemy`: Поведение противников (State Machine / Movement).
* `Combat`: Снаряды и расчет взаимодействий.
